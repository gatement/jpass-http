## configure
* edit `src/main/java/net/johnsonlau/jproxy/App.java`

## debug
* `./mvnw exec:java -Dexec.mainClass=net.johnsonlau.jproxy.App`

## package and run
* package `./mvnw clean package`
* run `java -jar target/jproxy.jar`
