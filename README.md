## run
* ./mvnw exec:java -Dexec.mainClass=net.johnsonlau.jproxy.App

## configure
* edit `src/main/java/net/johnsonlau/jproxy/conf/MyProxySettings.java`

## package
* ./mvnw  clean package
* java -jar target/jproxy.jar
